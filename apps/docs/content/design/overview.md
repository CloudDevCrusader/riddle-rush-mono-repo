# Design Overview

Riddle Rush features a modern, vibrant design system optimized for touch interactions and offline gameplay.

## Design Philosophy

The design emphasizes:

- **Touch-first**: Minimum 44px touch targets, large tap areas
- **Vibrant colors**: Orange primary, teal secondary with gradient backgrounds
- **Clear hierarchy**: Typography scale and spacing system
- **Accessibility**: WCAG AA compliant colors and keyboard navigation
- **Offline-first**: Visual feedback for offline states

## Design System Components

### Color Palette

**Primary Colors:**

- Primary Orange: `#FF6B35`
- Primary Light: `#FF8C61`
- Primary Dark: `#E65528`

**Secondary Colors:**

- Secondary Teal: `#4ECDC4`
- Secondary Light: `#7DD3CB`
- Secondary Dark: `#3AAFA9`

**Background Gradients:**

- Main Background: Purple gradient (`#667eea` → `#764ba2`)
- Warm Background: Pink gradient (`#f093fb` → `#f5576c`)
- Cool Background: Blue gradient (`#4facfe` → `#00f2fe`)

### Typography

- **Primary Font**: Inter (400, 500, 600, 700, 900)
- **Display Font**: Poppins (400, 500, 600, 700, 900)
- **Responsive sizing**: Fluid typography using `clamp()`

### Spacing System

8px grid-based spacing:

- XS: 4px
- SM: 8px
- MD: 16px
- LG: 24px
- XL: 32px
- 2XL: 48px
- 3XL: 64px
